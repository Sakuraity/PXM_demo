# 美妆包材独立站项目需求

## 项目背景
- **项目性质**：客户POC合作项目，复刻客户现有独立站
- **战略定位**：PXM Demo设计体系的重要组成部分，覆盖四大行业之一（美妆包材）
- **技术架构**：PXM系统 → 本地独立站后台 → 前端独立站

## 核心需求

### 1. 独立站复刻
- **视觉一致性**：与客户原网站外观保持完全一致
- **功能复刻**：复制现有网站的核心功能模块
- **用户体验**：给予客户熟悉的使用感受

### 2. 数据联通方案（重点）
- **数据链路**：PXM系统 → 本地后台 → 前端独立站
- **Demo展示**：在Demo中完整呈现数据流转过程
- **实时同步**：确保各层级数据的实时性和一致性

### 3. DIY定制模块（创新点）
- **入口设计**：产品详情页嵌入DIY入口
- **定制功能**：
  - 产品形状选择（瓶子、罐子等）
  - 颜色自定义
  - 设计图案上传
  - 3D预览效果
- **技术集成**：嵌入Realibox 3D配置功能
- **数据保存**：
  - 方案保存至心愿单
  - 同步至PXM系统
  - 供包材商跟进线索

## 技术实现要点

### 数据架构设计
```
PXM系统（底层数据）
    ↓ 数据接口
本地独立站后台（数据处理层）
    ↓ API接口
前端独立站（用户界面层）
```

### DIY模块技术栈
- **3D引擎**：Realibox 3D配置器
- **交互设计**：拖拽式操作界面
- **数据存储**：方案数据结构化存储
- **预览渲染**：实时3D效果预览
